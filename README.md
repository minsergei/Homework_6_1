Курсовая работа №5.

Проект для работы с API сайта hh.ru. 

По запросу пользователя получаем 10 работадателей(запрашиваемое слово должно содержаться в названии компании или описание).

Программа получит вакансии от 10 работадателей.

Программа создаст с помощью функций БД и таблицы в Postgres. Запишит полученные данные о работадателях и их вакансиях в таблицы.

Программа выведет с помощью класса DBManager sql-запросы. В последнем запросе пользователь должен ввести слово для поиска по вакансиям.
